# ARR Command Center

Proyecto para analizar, calcular, versionar y visualizar el ARR de isEazy a partir de datos de Salesforce y de la lógica histórica del Excel actual.

## Objetivo
Replicar y formalizar en una aplicación web la lógica actual de cálculo de ARR, incluyendo:
- ARR desde inicio de servicio
- ARR desde close won
- segmentación por líneas, consultor y otras dimensiones
- snapshots/versionado
- dashboard y reporting

## Método de trabajo
Este proyecto sigue un enfoque Spec-Driven Development y está diseñado para continuidad entre Claude Code y Codex.
